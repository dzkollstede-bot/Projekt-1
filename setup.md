# 🔒 Secure Boot Configuration

> ⚠️ BIOS layouts differ depending on manufacturer

---

## 🧩 Quick Overview

| Manufacturer | BIOS Path | Setting |
|-------------|----------|--------|
| ASUS | Boot → Secure Boot | OS Type → Other OS |
| MSI | Settings → Advanced → Windows OS Configuration | Secure Boot → Disabled |
| Gigabyte | BIOS → Secure Boot | Disabled |
| ASRock | Security → Secure Boot | Disabled |
| Lenovo | Security → Secure Boot | Disabled |
| HP | System Configuration → Boot Options | Disable Secure Boot |
| Dell | Boot Configuration | Secure Boot → Disabled |

---

## 🖥️ Detailed Steps

### 🔧 ASUS
| Step | Action |
|------|--------|
| 1 | Enter BIOS |
| 2 | Press F7 (Advanced Mode) |
| 3 | Navigate to Boot → Secure Boot |
| 4 | Set OS Type → Other OS |

---

### 🔧 MSI
| Step | Action |
|------|--------|
| 1 | Enter BIOS |
| 2 | Go to Settings |
| 3 | Advanced → Windows OS Configuration |
| 4 | Disable Secure Boot |

---

### 🔧 Gigabyte
| Step | Action |
|------|--------|
| 1 | Enter BIOS |
| 2 | Go to BIOS tab |
| 3 | Open Secure Boot |
| 4 | Disable it |

---

### 🔧 ASRock
| Step | Action |
|------|--------|
| 1 | Enter BIOS |
| 2 | Go to Security |
| 3 | Open Secure Boot |
| 4 | Disable it |

---

### 💻 Laptops (General)
| Brand | Path |
|------|------|
| Lenovo | Security → Secure Boot |
| HP | System Configuration → Boot Options |
| Dell | Boot Configuration |

---

## ✅ Final Check

| Check | Status |
|------|--------|
| Secure Boot Disabled | ✔ |
| System boots normally | ✔ |
| No BIOS errors | ✔ |

---

> 💡 Tip: BIOS versions may differ slightly depending on model.
